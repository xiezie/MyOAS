## 框架

## 环境
Eclipse Java EE IDE|Kepler Service Release 2
数据库|M0ySql 5.0.22
Web容器|Tomcat 6.0
操作系统|Windows XP

## 约定
* 工程中所有的文件都采用utf-8编码。所有的页面都是使用utf-8编码，这样方便解决表单内容的乱码问题。乱码一般情况下是乱用编码产生的。
* 实体的主键属性的类型使用Long型。
